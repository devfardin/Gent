# Gent

Gent is a custom WordPress and WooCommerce management plugin developed for the Gents Time fashion brand. It centralizes business settings, customer management, order automation, WhatsApp integration, and advanced eCommerce functionality through a dedicated admin dashboard.

## Features

* Global website settings management
* Social media and contact information management
* WhatsApp integration and automation
* Duplicate order detection and prevention
* Customer blocking and restriction system
* Custom WooCommerce order statuses
* Incomplete order tracking
* Popup and notification manager
* Customer insights and analytics
* Order activity logs
* Extensible architecture for future modules

## Planned Features

* Abandoned cart recovery
* Loyalty and rewards system
* Wishlist management
* Referral program
* Advanced customer segmentation
* Marketing automation

## Purpose

The plugin separates business logic from theme design, making the system maintainable, reusable, and scalable for future eCommerce projects.
